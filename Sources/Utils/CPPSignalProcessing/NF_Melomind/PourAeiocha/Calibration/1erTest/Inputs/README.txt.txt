################# trueCalibrationRecordings ########################
trueCalibrationRecordings contains the EEG values of the calibration.
This is an array of 2 rows (2 channels) and 250x30 columns (EEG values).

################## trueCalibrationRecordingsQuality ########################
trueCalibrationRecordingsQuality contains the qualities of the EEG packets
of 1s during calibration. This is an array of 2 rows (2 channels) and 30
columns.
