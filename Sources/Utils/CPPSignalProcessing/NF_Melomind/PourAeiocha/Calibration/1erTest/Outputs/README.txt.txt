################# BoundsCalib ########################
BoundsCalib contains the boundaries to detect outliers. 
This is an 2x2 array because 2 bounds for each channel.

################## bestChannel########################
bestChannel contains the best channel (based on the 
study of the qualities)

################### SNRCalib #########################
SNRCalib contains the relaxation index of the calibration
of the best channel.
There are 27 values for 30s of calibration.
