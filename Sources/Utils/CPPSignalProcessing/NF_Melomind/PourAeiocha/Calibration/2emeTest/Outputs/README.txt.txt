################# BoundsCalib ########################
BoundsCalib contains the boundaries to detect outliers. 
This is an 2x2 array because 2 bounds for each channel.

################## bestChannel########################
bestChannel contains -2 because both channels have bad
qualities.

################### SNRCalib #########################
SNRCalib contains only inf because the calibration 
contains bad qualities.

An error message should appear in the console:
ERROR: MBT_COMPUTECALIBRATION CANNOT PROCESS WITH ONLY BAD QUALITY SIGNALS
