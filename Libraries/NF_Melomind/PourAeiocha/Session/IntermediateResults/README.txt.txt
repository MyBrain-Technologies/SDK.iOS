######################## SNRSession ################################
SNRSession contains the SNR values of the session of
the best channel (output MBT_ComputeRelaxIndex).

###################### SmoothedSNRSession ###########################
SmoothedSNRSession contains the smoothed SNR values of the
session (output of MBT_SmoothRelaxIndex).

################# NormalizedSmoothedSNRSession ######################
NormalizedSmoothedSNRSession contains the normalized smoothed SNR 
values of the session (output of MBT_NormalizeRelaxIndex).
